How to Fix Your Main Website

To make the font work on your main website, you need to:

  1. Add the @font-face code to your CSS file
  2. Apply the font to elements in your CSS
  3. Make sure the font files are in the correct location
/* Add this to your style.css file */
@font-face {
    font-family: 'Ebrima';
    src: url('fonts/ebrima.ttf') format('truetype');
    font-weight: normal;
    font-style: normal;
}

@font-face {
    font-family: 'Ebrima';
    src: url('fonts/ebrima-bold.ttf') format('truetype');
    font-weight: bold;
    font-style: normal;
}

/* Apply the font to elements */
body {
    font-family: 'Ebrima', sans-serif;
}

Note: Make sure your font files are in a folder named "fonts" in your main directory, and that the filenames match exactly (including capitalization).
